介绍：
这是一个可以在Minecraft中的kubejs框架下执行的脚本，其作用是生成一个中式传统中小型硬山顶建筑。

Introduction:
This is a script can run in kubejs, Minecraft. Its function is to generate a traditional Chinese medium&small-sized gable roof building.

使用说明：
你首先需要一个安装了Kubejs的Minecraft版本，然后将generate.js放到存档文件夹中的kubejs\server_scripts中。
你可以随意地修改该文件开头的数值和方块id（建议使用VS Code）。
启动游戏后，当你使用钻石斧右键后，便可以在世界的大概(0, 0, 0)处找到生成的建筑。

Guide to Use:
You need a Minecraft version where the mod kubejs is installed, then put generate.js file into a folder in the folder of your minecraft version, kubejs\server_scripts.
You can freely changed the values and id of blocks at the head of the script (Recommend to use VS Code).
After luanching your game, when you hold a diamond axe and right click, you can find the building near (0, 0, 0) in the world.
